[MySQL] MySQL重置密碼及修改密碼

本文最後更新於：2024年3月5日 晚上

此文章記錄在Linux上 MySQL重置密碼及修改密碼方式

重置密碼

  1. 使用指令找到MySQL讀取的my.cnf檔案路徑：

    mysql --help | grep 'Default options' -A 1

  2. 編輯my.cnf

    vim [檔案路徑]

    在[mysqld]標籤底下加入 skip-grant-tables(註1)

  3. 使用指令重啟資料庫

    service mysql restart

  4. 登入MySQL(直接輸入Enter不需要打密碼)，並使用以下指令(註2)

    use mysql;
update user set authentication_string = '' where user = 'root';
FLUSH PRIVILEGES;
EXIT

  5. 重新編輯my.cnf，並將剛剛加入的skip-grant-tables 移除

  6. 再次重啟資料庫

  7. 即可使用空字串作為密碼進行登入

修改密碼

修改密碼相當簡單，使用有權限的帳號登入MySQL後，使用以下指令：

use mysql;
SELECT * FROM user WHERE user LIKE 'root%' \G;
ALTER USER root@localhost IDENTIFIED WITH mysql_native_password BY 'newPassword';
FLUSH PRIVILEGES;
EXIT

環境：

  • Ubuntu 16.04.6 LTS
  • MySQL 5.7.33
